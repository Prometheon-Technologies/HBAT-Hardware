# Rack Mounted Design

This is a design for a rack mounted server.  It is designed to be a modular and extensible design.

## Design

### Electrics

#### Power

- 120A breaker (x1)
- M8 terminal block (x4)
- Power Button (x1)
- RST Button (x1)
- Power Indication LED (x1)
- Charge Indication LED (x4)

#### Wiring

- RJ45 for signals
- RS232 for debug
- RS485A CAN to inverter
- RS485B Internal COM
- RS485C Internal COM
